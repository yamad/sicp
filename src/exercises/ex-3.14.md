> Exercise 3.14
>
> Explain what the following code does:
>
> ```scheme
> (define (mystery x)
>   (define (loop x y)
>     (if (null? x)
>         y
>         (let ((temp (cdr x)))
>           (set-cdr! x y)
>           (loop temp x))))
>  (loop x '()))
> ```

`mystery` reverses list `x` in place. It uses no extra memory other
than one instance of the temp variable.

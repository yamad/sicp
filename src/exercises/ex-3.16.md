> Exercise 3.16
>
> ```scheme
> (define (count-pairs x)
>   (if (not (pair? x))
>       0
>       (+ (count-pairs (car x))
>          (count-pairs (cdr x)))
>          1))
> ```
>
> Explain why this procedure is not correct.

(count-pairs a) would return 3
```
         +---+---+    +---+---+    +---+---+
  a ---> | . |  ----->| . |  ----->| . | / |
         +---+---+    +---+---+    +---+---+
```

(count-pairs b) would return 4
```
         +---+---+    +---+---+     +---+---+
  b ---> | . |  ----->| . |  ------>| . | / |
         +---+---+    +-|-+---+     +---+---+
                        |             ^
                        .-------------|
```

(count-pairs c) would return 7
```
         +---+---+     +---+---+     +---+---+
  c ---> | . |  ------>| . |  ------>| . | / |
         +-|-+---+  |  +-|-+---+  |  +---+---+
           |        |    |        |
           .--------.    .--------.
```

(count-pairs d) would never return
```
         +---+---+     +---+---+     +---+---+
  d ---> | . |  ------>| . |  ------>| . | . |
     ^   +-|-+---+  ^  +-|-+---+  ^  +---+-|-+
     |     |        |    |        |        |
     |     .--------.    .--------.        |
     |                                     |
     .-------------------------------------.
```
